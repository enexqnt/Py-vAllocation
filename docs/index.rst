Py-vAllocation documentation
============================

Py-vAllocation is a Python package for asset allocation with a focus on integrating investor views. The documentation provides an overview of the API and usage examples.

For practical examples and Jupyter notebooks, please refer to the `examples directory on GitHub <https://github.com/giuliocantadori/Py-vAllocation/tree/main/examples>`_.
